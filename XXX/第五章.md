
## 運算子的運用1
```
INPUT輸入:一個正整數
a=2;

OUTPUT輸出:印出a等於多少
printf("a=%d\n",a++); 
printf("a=%d\n",++a); 
printf("a=%d\n",a--); 
printf("a=%d\n",--a); 

以上輸出答案為多少?如果答案有差異請說明
```

## 運算子的運用3
```
INPUT輸入:二個正整數
a=2;
b=3;

OUTPUT輸出:印出c等於多少
a=b;
printf("c=%d\n",a+b); 
printf("c=%d\n",a-b); 
printf("c=%d\n",a*b); 
printf("c=%d\n",a/b); 
printf("c=%d\n",a%b); 
printf("c=%d\n",!(a+b)); 

以上輸出答案為多少?
```
## 運算子的運用4
```
INPUT輸入:二個正整數
a=2;
b=3;

OUTPUT輸出:
if(a*b>6 || a/b<1)
      printf("A");
      
 if(a*b>6 && a/b<1)
      printf("B");
      
 if(a++>1 && ++a>3)
      printf("C");
      
 if(a++>4 || ++a>5)
      printf("D");    
 
 printf("a=%d",a);
 
 
以上輸出答案為多少?解:ACDa=6
```
